# 實戰案例:B2 青銅武將像 — 物件合成進場景(指定位置 + 對齊光影)

> 一張森林場景圖 + 一張去背物件圖 → 把物件放進指定位置、鎖住大小,
> 並讓物件的材質/光照/反射自動對齊環境,自然融入。
> 用來示範 [../references/nano-banana.md](../references/nano-banana.md#物件合成進場景-game-asset--把物件放進背景圖--實測通用模板) 的物件合成模板。

| 項目 | 內容 |
|---|---|
| 類別 | **[B] 物件合成進場景**(把既有資產放進背景板、對齊環境光影) |

## 用到的 Prompt(逐字保存 — 實測通用版)

```text
請參考圖1的基本構圖（只參考構圖，依需求做調整，並重繪一張），
使用圖2作為地上置放雕像的圖案（請使用圖2的圖案）

需求：

game art background asset

將圖2置放於畫面中，不改變圖2原來的大小跟位置
調整材質符合周圍的材質表現，符合周圍的光照、反射、漫反射
```

- **圖1 = 場景圖**(構圖參考,允許重繪)
- **圖2 = 去背物件圖**(造型來源,image-grounded,不靠文字描述外觀)

## 心法與結論(可複用)

1. **少寫 > 逐條指定。** 不要硬寫 cast shadow 角度 / rim light 方向 / AO 強度——容易跟畫面打架且脆。改成只下「材質與光照向周圍對齊(光照、反射、漫反射)」的**整包**指令,讓模型用世界知識自己算,融入度反而更高、更通用。
2. **「重繪一張」是融入感關鍵。** 允許模型連背景帶物件一起重畫,光影是「一次算出來的一致場景」,不是「貼上去再補影子」。
3. **換圖1 = 多場景變體。** 同物件、同 prompt,只換場景圖,模型自動把物件重打光成那個調性(暖陽 vs 冷霧長苔),連青苔覆蓋、材質鏽蝕都跟著環境變——做一套資產的多場景變體超快。
4. **兩種策略要分清:**
   - 要**極致融入**(物件可被環境改寫,長苔、鏽蝕)→ 用本案這套「重繪」法。
   - 要**保留物件原貌**(商品 / IP 不能變形)→ 改用 `nano-banana.md` 的「鎖死物件 only relight」那套。
